## What it does

- Renders LaTeX equations
- Export to PNG
- Share equations via URL
- Works offline (PWA)
- Client-side only, no server storage

## Setup

Built with Svelte, Vite, Tailwind CSS, and KaTeX.

## Running it

```bash
cd app
npm install
npm run dev
```

Open `http://localhost:5173`
